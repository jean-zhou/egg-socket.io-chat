# egg-socket.io-chat

# 使用 express 搭建后台和html搭建前端 —— 可以链接socket

## 前端使用 socket.io 链接 socket 后台
项目文件：client-html
1，直接使用html即可
2，引入 socket.io 库
3，链接后端端口 io.connect('http://127.0.0.1:3000')

## 使用 express+ socket.io 搭建socket后台
项目文件：express-socket
1，安装 express + socket.io
2，创建index.js 链接 socket
